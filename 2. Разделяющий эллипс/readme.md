 У нас есть два класса объектов в двумерном пространстве. Точки первого класса X1 разбросаны вокруг центра μ1 с "формой" разброса, заданной матрицей Σ1. Аналогично для второго класса X2.

Мы ищем границу между этими классами. На этой границе вероятность того, что точка x принадлежит первому классу, равна вероятности, что она принадлежит второму классу: P1(x) = P2(x).

В случае нормальных распределений, эта граница (при равенстве ковариационных матриц) была бы прямой линией. Но так как матрицы Σ1 и Σ2 разные, эта граница превращается в кривую второго порядка, которая в данном случае будет эллипсом.
